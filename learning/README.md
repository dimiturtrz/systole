# learning/

My study material + quiz logs for the systole project — my ramp into cardiac imaging. One file per topic:
the **theory writeup** (what I study / prepare for), then a dated **quiz log**
(questions, my answers, honest assessment) appended when I ask to be quizzed.

These are the grid's "theory" cells (see [../docs/ROADMAP.md](../docs/ROADMAP.md)) —
learning-in-public, public at Gate 1.

## The circuit (how each topic is learned)
1. **Research** — teacher grounds the topic (internal knowledge + web) →
   `../research/` (tracked, public; cited raw material behind the writeups).
2. **Theory** — teacher writes the study writeup here → `<date>_<topic>.md`.
3. **Quiz (on demand)** — when I say *"quiz me,"* open-form questions on that theory.
4. **Log** — my answers + honest assessment + score appended to the topic file.
5. **Sharpen** — update the ROADMAP grid cell + set the next concrete step.

**Honest by rule:** domain facts are source-grounded, not asserted from memory;
wrong answers stay in the log — the ramp is real, the self-eval is honest.
