### Q1 · Why the NIfTI affine is 4×4 not 3×3 — 5/5
**Answer:** linear combos plus a bias term — a fixed shift.
**Grade:** ✓ The 4th row/col adds **translation** (homogeneous coords); a 3×3 fixes the
origin (rotate/scale/shear only).

### Q2 · Why a deep conv stack ≠ one conv — 3.5/5
**Answer:** (hedged) the activation breaks linearity? something about residuals?
**Grade:** Found it — **the nonlinearity.** Without it, composed linear maps collapse to
one linear map → depth buys nothing. (Residuals = distractor.)

### Q3 · Resample 1.56→1.50 mm, 256 px → ? — 4.5/5
**Answer:** 256 / (1.5/1.56).
**Grade:** ✓ = 256 × 1.56/1.5 ≈ **267** (finer pixels → more of them). Land the number.

### Q4 · Why backprop is right-to-left / what's reused — 2.5/5
**Answer:** computes loss back the layers, fixes weights; a mistake is judged after it happens.
**Grade:** Got *why start at the right* (need the loss first). Missed **what's reused**:
each layer reuses the **downstream gradient** (accumulated chain-rule product), computed
once and shared — left-to-right would recompute the whole chain per weight.

### Q5 · NaN loss → which hyperparameter — 5/5
**Answer:** learning rate, down; like RNN gradient explosion.
**Grade:** ✓ LR too high → overshoot → diverge. Same family as gradient explosion.

### Q6 · Softmax + constant to all logits — 1/5  ⚠ inverted
**Answer:** it changes if we add (not multiply); not called anything.
**Grade:** Backwards. Adding a constant does **NOT** change softmax — **shift-invariance**
(the e^c cancels top and bottom). What changes it: **scaling** (temperature).

### Q7 · What cross-entropy secretly maximizes — 1.5/5  ⚠
**Answer:** MSE but for classification.
**Grade:** It's **maximum likelihood** — minimizing CE maximizes the **probability assigned
to the true label**. (MSE is also MLE, but for Gaussian/regression; CE is MLE for
categorical/classification — so the analogy is loosely apt, not the answer.)

### Q8 · Imbalance → predict background everywhere — 3.5/5
**Answer:** don't know the term; high accuracy, low F1; Dice normalizes error by the opportunity.
**Grade:** Symptom ✓ (high acc, low F1). Term = **biased/degenerate estimator** (collapses
to majority). Dice **normalizes to the foreground** → all-background scores ~0 → trivial
solution punished. ✓ mechanism.
